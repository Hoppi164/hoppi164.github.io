## 1. Install

```bash
git clone git@github.com:Hoppi164/hoppi164.github.io.git
```

```bash
cd hoppi164.github.io
```

```bash
npm install
```

<br>

## 2. Develop

To start the dev server

```bash
npm run dev
```

If using netlify-cms to edit posts:

```bash
npm run build && npm run preview
```
